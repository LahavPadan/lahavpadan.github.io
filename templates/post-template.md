## The first idea

Use ordinary Markdown. Obsidian previews standard Markdown images too:

![A useful diagram](diagram.svg)

For inline math, write $a^2+b^2=c^2$.

For display math, give the equation its own paragraph:

$$
\nabla f(\mathbf{x}) = 0.
$$

Put `diagram.svg`, `figure.png`, or any other images in this same folder as `index.md`.
