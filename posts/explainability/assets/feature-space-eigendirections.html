Reading the Gram matrix

A near-dependency is a narrow direction in feature-space

The same sample cloud can be broad in one eigendirection and almost collapsed in the perpendicular one. The eigenvalue records that projected spread.

weakstrong
λᵢ = ‖Xqᵢ‖² for a unit eigenvector qᵢ, after the feature columns are centered
q₁

Broad projected spread

The samples differ substantially along this combination of features, so λ₁ is large.

q₂

Tight projected cluster

The corresponding column combination nearly cancels, so Xq₂ and λ₂ are small.

Near-dependence: the cloud has not disappeared; it has become thin in one particular direction. That thin direction is the unstable eigenvector.
