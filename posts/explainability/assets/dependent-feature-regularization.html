Geometric intuition

Linearly dependent features: why ℓ1 selects and ℓ2 spreads

If one feature is a scalar multiple of another, the loss depends only on one linear combination of the coefficients. The minimizers therefore form a line of equal loss.

x2 = c x1 ⇒ β1 + cβ2 = s

01 ℓ1 penalty

Expand the diamond until it first reaches the equal-loss line.

1

The first contact with the diamond occurs at a vertex.

2

That vertex lies on an axis, so one coefficient is exactly zero.

02 ℓ2 penalty

Expand the circle until it first reaches the same equal-loss line.

1

The first contact with the circle occurs away from the axes.

2

Both coefficients remain nonzero, so the fitted weight is shared.
