Current routing

P1 → P2, with P3 isolated.

P3 lies exactly on the nodal diameter, so E(P3) = 0. The remaining neighboring port has nonzero overlap with the rotated dipole.

Why the required rotation is 30°

Port spacing φtarget − φin = 120°
Dipole geometry φnull − φa = 90°
Required shift α = 120° − 90° = 30°

Rotating the antinode axis by +30° rotates the nodal diameter by the same +30°. Reversing the bias changes α to −30°, so the other neighboring port lies on the null.

Same bias, all three inputs

+B₀ fixes α = +30° for every input. Rotating the entire construction by 120° gives the cycle P1 → P2 → P3 → P1.
