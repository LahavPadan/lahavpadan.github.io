B

Why the determinant stays equal to one

Start again from the scalar wave equation and the already-defined state v = ( E E′ ) . The matrix below is not assumed; its two rows are just the two component equations written together.

Start from the scalar equation. It gives the second component derivative explicitly.
E″ + ( ωc ) 2 ε(z) E = 0
top
d dz E = E′ = 0·E + 1·E′.

This is why the first row is (01).

bottom
d dz E′ = E″ = − ( ωc ) 2 ε(z) E + 0·E′.

This is why the second row is ( − ( ωc ) 2 ε(z) 0 ) .

The two component equations, written together

d dz ( E E′ ) = ( 0 1 − ( ωc ) 2 ε(z) 0 ) ( E E′ ) .
1

Differentiate the transfer relation

v(z) = T (z0,z) v (z0).
v′ = T′ v (z0).

The first-order equation just derived also gives

v′ = ( 0 1 − ( ωc ) 2 ε(z) 0 ) T v (z0).

These expressions agree for every initial state, hence

T′ = ( 0 1 − ( ωc ) 2 ε(z) 0 ) T.
2

Write the resulting entry equations

T = ( a b c d ).

For compactness only, set

r(z) ≡ ( ωc ) 2 ε(z).
a′=c , b′=d , c′=−ra , d′=−rb.
3

Differentiate the determinant

detT = ad − bc.
d dz detT = a′d + ad′ − b′c − bc′
= cd − rab − dc + rba = 0.

Thus the determinant is constant in z. At zero propagation distance, T (z0,z0) = I, so that constant is one.

The derivative vanishes and the initial map is the identity
d dz detT =0 , detT (z0,z0) =1.
⇒
det T (z0,z) = 1.
This is the fact used in § 8.3. For the two eigenvalues of one cell, λ1 λ2 = det Tcell =1.
